# Express Book Reviews API

A RESTful API built using Node.js and Express.js that allows users to browse books, register/login, and add or manage reviews. This project also demonstrates authentication using JWT, session handling, and asynchronous API calls using Axios.


## Features

### General Users
- Get all books
- Get book by ISBN
- Get books by Author
- Get books by Title
- Get book reviews

### Registered Users
- User Registration
- User Login (JWT + Session)
- Add / Modify Book Review
- Delete Book Review

### Advanced Features
- Authentication using JWT
- Session management using express-session
- Async/Await & Promises using Axios


##  Tech Stack

- Node.js
- Express.js
- JSON Web Token (JWT)
- express-session
- Axios
